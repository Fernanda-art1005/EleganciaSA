# Elegância Premium SA - Painel Comercial

Este é o sistema oficial de gestão e painel comercial da Elegância Premium SA, desenvolvido com uma interface moderna, rápida e responsiva.

## 🚀 Tecnologias Utilizadas
- **React 19** (Interface declarativa e componentes modulares)
- **Vite 6** (Ambiente de desenvolvimento e empacotamento ultra rápido)
- **Tailwind CSS 4** (Estilização moderna e utilitária)
- **Lucide React** (Pacote de ícones bonitos e leves)
